# §3.2 Literature Review — LEFT ARM Template

> **Horseshoe role:** Establish the gap that the bridge (Methodology) will
> fill. Partner paragraph: Conclusion ¶ 3 (contributions to knowledge).

## Funnel Structure (3 subsections, broad → narrow)

### §2.1 Broad Domain

Establish the general field. ~1 paragraph. Cite 4–8 papers.

```
{BROAD DOMAIN} has attracted significant research attention due to
{MOTIVATION}. Early work focused on {EARLY APPROACH} (Author Year; Author
Year). More recently, {RECENT DIRECTION} has emerged as a promising
direction (Author Year; Author Year).
```

### §2.2 Specific Task Family

Narrow to the immediate neighborhood of the contribution. ~2–3 paragraphs.
Organize by **approach class** (e.g., classical ML vs. deep learning, or
vision-based vs. acoustic).

```
Within {SPECIFIC TASK FAMILY}, prior methods can be grouped into {N}
categories. {CATEGORY 1}: {SUMMARY + KEY CITATIONS}. {CATEGORY 2}: {SUMMARY +
KEY CITATIONS}. ...
```

### §2.3 The Exact Gap (1 paragraph)

End with the precise gap that the paper fills. This is the **pivot** into
§Methodology.

```
Despite this progress, {SPECIFIC UNADDRESSED PROBLEM}. Specifically, {GAP
DETAIL 1}. {GAP DETAIL 2}. {OPTIONAL: "To the best of our knowledge, no
prior work has addressed {X}."} This paper addresses these gaps by
{ONE-SENTENCE METHOD PREVIEW, expanded in §3}.
```

## Citation Conventions (Multi-Venue — see SKILL.md §4)

- **ASCE / Elsevier author-year**: `(Author and Author Year)` — no comma
  before year.
- **IEEE / MDPI numbered**: `[1]`, `[2]`, … in order of appearance.
- **Three or more authors**: `et al.` after first full-mention.
- Always prefer the **original** paper for an idea, not a survey that cites
  it.
- Cite prior work on the same task or by the same research group only where
  **genuinely relevant** — never gratuitously.

## Self-Check
- [ ] Funnel narrows visibly: §2.1 broadest, §2.3 narrowest.
- [ ] §2.3 ends with a clear pivot sentence pointing to §3 Methodology.
- [ ] No citation dump — every cited paper has a one-line characterization
      (not just "(Author Year; Author Year; Author Year)").
- [ ] Gap is **specific**, not "more research is needed".
- [ ] Citation format matches the target venue style (ASCE / Elsevier /
      IEEE / MDPI / Springer).

## Mirror Reminder
§2.3's gap statement maps to Conclusion ¶ 3 (contributions to knowledge).
The Conclusion will say *"This paper contributes to {FIELD} by {FILLING THE
GAP DEFINED HERE}."*
