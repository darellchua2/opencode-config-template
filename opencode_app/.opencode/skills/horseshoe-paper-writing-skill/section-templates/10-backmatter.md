# Back-Matter Template — Data Availability, Acknowledgments, CRediT

> **Horseshoe role:** Required by ASCE and most Elsevier venues. Not part of
> the horseshoe arms or bridge — these are **metadata** sections that follow
> the Conclusion.

## Data Availability Statement

ASCE / Elsevier / MDPI require this; IEEE / Springer recommend it. Use one
of the standard phrasings:

### Fully open data
```
All data, models, and code that support the findings of this study are
available from the corresponding author upon reasonable request, and in the
public repository: {REPO URL} (DOI: {ZENODO / FIGSHARE DOI}).
```

### Partially available
```
Some or all data, models, or code that support the findings of this study
are available from the corresponding author upon reasonable request.
{SPECIFY WHAT IS AND ISN'T AVAILABLE: e.g., "Raw audio recordings are
restricted due to {PRIVACY / IP}; extracted features and trained model
weights are publicly available at {URL}."}
```

### Available on request only (most common default)
```
Some or all data, models, or code that support the findings of this study
are available from the corresponding author upon reasonable request.
```

### Embargoed
```
Data and code will be made publicly available upon publication of the
associated press release / patent / {DATE}. Until then, they are available
from the corresponding author upon reasonable request.
```

## Acknowledgments

```
The authors gratefully acknowledge the financial support of {FUNDING AGENCY}
under Grant {NUMBER}, and {OTHER FUNDING}. The authors thank {INDIVIDUALS}
for {DATA COLLECTION / DISCUSSION / REVIEW}. {OPTIONAL: "We acknowledge the
use of {AI TOOL: ChatGPT / GitHub Copilot / etc.} for {PROOFREADING / CODE
GENERATION}; all scientific content was verified by the authors."}
```

**Common funder categories:**
- National / university funding agencies (e.g., NRF, NSF, EPSRC, JSPS)
- Research councils under strategic programs (e.g., CREATE-type campus
  programs, ERC grants)
- Industry partners (disclose any conflict of interest explicitly — see
  below).
- Internal university seed grants.

## Author Contributions (CRediT Taxonomy)

ASCE / Elsevier / MDPI require the **CRediT** statement. Assign all 14
roles.

```
{AUTHOR 1 NAME}: Conceptualization; Formal analysis; Investigation;
Methodology; Validation; Writing – original draft; Writing – review & editing.

{AUTHOR 2 NAME}: Conceptualization; Methodology; Validation; Writing –
review & editing.

{AUTHOR 3 NAME}: Conceptualization; Data curation; Investigation; Writing –
original draft.

{AUTHOR 4 NAME}: Conceptualization; Data curation; Formal analysis;
Methodology; Writing – original draft.

{PI / CORRESPONDING AUTHOR}: Conceptualization; Data curation; Formal
analysis; Investigation; Methodology; Supervision; Writing – original
draft; Writing – review & editing.
```

### CRediT Roles (all 14)

| Role | Meaning |
|------|---------|
| Conceptualization | Ideas; formulation of research goals |
| Methodology | Development or design of methodology |
| Software | Programming, software design |
| Validation | Verification of results / reproducibility |
| Formal analysis | Statistical, mathematical, computational |
| Investigation | Conducting experiments / data collection |
| Resources | Provision of study materials, equipment |
| Data curation | Management, annotation, maintenance of data |
| Writing—original draft | Initial draft of the manuscript |
| Writing—review & editing | Critical review, commentary, revision |
| Visualization | Figures, diagrams, data presentation |
| Supervision | Oversight, mentorship, group leadership |
| Project administration | Management, coordination |
| Funding acquisition | Financial support for the project |

**PI / corresponding author pattern (typical):** Conceptualization +
Methodology + Supervision + (Funding acquisition) + Writing—review & editing.
**Sometimes also:** Data curation, Formal analysis, Investigation,
Writing—original draft (if hands-on involved).

## Conflict of Interest

If any author has a financial relationship with an industry partner (e.g.,
founder / advisor / shareholder of a company with an interest in the
research), disclose:

```
{AUTHOR NAME} is a {FOUNDER / ADVISOR / SHAREHOLDER} of {COMPANY}, which
{RELATIONSHIP TO RESEARCH}. The remaining authors declare no conflicts of
interest.
```

If none: `The authors declare no conflicts of interest.`
